The World Wide Web & HTML

Important Concepts

World Wide Web

  • A collection of HTML documents made up of HyperText Markup Language
  • Everything found online (docs., images, videos) is glued together by HTML
  • Hyper Links connect all the HTML documents together making the World Wide Web, web-like
How It Works

I open my favorite browser and and type in www.Udacity.com and hit go. My browser sends a request via the Internet using a protocol called HTTP to the servers hosting Udacity's files. The servers respond with the requested files, which my browser then displays. For more information, check out this video.

HTML

HTML is made up of 4 important components:

  1. Text Content (The actual text you see.)
  2. Markup (How the text looks, color, arangment, etc.)
  3. References to Other Documents (Images, Videos, Games etc.)
  4. Links to Other Pages (Hence, the web-like structure.)
HTML Markup

HTML Markup is made up of tags. These tags tell the computer how to display the text. A tag usually consists of both an opening and a closing tag, with the contents in between, and the entire entity is referred to as an element. Some tags, like the image tag, do not contain any text content to be displayed and therefore do not have an end tag. A tag can have none, one, or multiple attributes inside of it. This Video gives a good overview.

Inline v.s. Block Elements

HTML elements can be either inline, or block. Block elements form an invisible box around the contents inside of them. This box can be adjusted in both height and width. Inline elements simply appear in line with the text.

Think Like A Programmer

Thinking like a programmer involves:

  1. Procedural Thinking:

    2. Taking a big problem and breaking the solution down into a set of steps.

  2. Abstract Thinking:

    3. Creating generalizations or finding similarities between seemingly unrelated things.

  3. Systematic De-Bugging:

    4. Systematically identifying problems and tracking down their solution.

  4. Technological Empathy:

    5. Understanding how a computer sees things.

  5. Systems Thinking:

    6. Taking a big idea and breaking it down into its component pieces.
